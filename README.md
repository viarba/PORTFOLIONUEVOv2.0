# MI PORTFOLIO DE DESARROLLO WEB

Proyecto interactivo de portfolio estructurado en una única página interactiva.

## 🛠️ Tecnologías Utilizadas
* HTML5 / CSS3 (Diseño responsivo con Media Queries)
* JavaScript / SweetAlerts2 (Validación dinámica y gestión de eventos de teclado y ratón)
* PHP (Procesamiento y validación segura en el lado del servidor)

## 🚀 Requisitos de Despliegue
* **Interfaz estática (HTML, CSS, JS):** Puede ser visualizada y desplegada en **GitHub Pages**.
* **Procesamiento de Formulario (PHP):** Requiere un entorno de servidor como Apache (XAMPP / WampServer) o hosting compatible con PHP en producción.

## ⌨️ Atajos del sitio
* Presiona la tecla `C` en cualquier momento para ir directamente al formulario de contacto.
